About Me

Hi, I am Mohsin Sajan, a QA turned self taught Full Stack Developer from India.

In my second year working as a QA I started learning Python for Automation and I found that I actually liked programming. I realized that I actually enjoyed the process of creating something of my own and thus started my journey into the programming world. I started learning HTML,CSS and JS and started doing small projects like a Todo App and such. But it wasn't until 2020 that I decided to seriously consider moving into Web Development and started learning and building some big projects.

So far into my journey I have learnt HTML CSS JS React and Django. Django was the first framework I learnt and I fell in love with it, hence I decided to improve my Django knowledge and learn how well I can integrate Django with React(another framework that I absolutely love).
